HTML

Front-end Technology

HTML is the standard markup language for creating web pages. HTML consists of HTML tags, which define the structure and content of a web page. HTML is used to create hypertext documents with the aim of making them easy to read and navigate.

CSS

Front-end Technology

Css is a stylesheet language used for describing the presentation of a document written in HTML. CSS is designed to make web pages more dynamic and user-friendly. CSS allows you to style HTML elements to create visually appealing web pages.

JavaScript

Front-end Technology

JavaScript is a scripting language that adds interactivity to web pages. JavaScript allows you to create dynamic and interactive web pages. JavaScript can be used to add functionality to HTML elements, change the content of HTML elements, and perform client-side validation.

Java

Back-end Technology

Java is a general-purpose programming language that was developed by Sun Microsystems. Java is designed to be concise, secure, and object-oriented. Java is used to create applications that require a high degree of interactivity, security, and scalability.

MongoDb

Database

MongoDB is a NoSQL database that stores data as JSON-like documents. MongoDB is designed to handle large amounts of data efficiently and scale to multiple servers. MongoDB is used to build applications that require a high degree of data storage, retrieval, and modification.

ReactJs

JavaScript library

ReactJS is a JavaScript library for building user interfaces. ReactJS is used to create single-page applications (SPAs) that are easy to understand, maintain, and scale. ReactJS is used to build applications that require a high degree of interactivity, user experience, and scalability.
